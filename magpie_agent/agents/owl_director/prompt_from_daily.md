### 정기 검진 모드 지침 (매일 오전 9시)

매일 오전 9시 정기 검진(DailyReport) 모드로 실행되었습니다.
다음 절차에 따라 오늘의 투자 포트폴리오를 점검하고 갱신하세요.

1. **매매 이력 분석**
   - 최근 24시간 동안의 체결 이력을 확인하세요.
   - 익절/손절이 발생했다면 그 원인을 분석하세요.
   - 현재 보유 중인 포지션의 손익 상태를 파악하세요.

2. **전략 적합성 평가**
   - 현재 등록된 전략이 최신 시장 상황에 여전히 적합한지 판단하세요.
   - 전략이 유효하다면 `transfer_to_agent("meerkat_scanner")`를 호출하여
     기존 종목에 대한 타점만 갱신하세요.
   - 전략 수정이 필요하다면 새로운 전략 초안을 사용자에게 보고하지 말고
     직접 `register_strategy_to_nest`를 호출하여 전략을 갱신하세요.
     (정기 검진이므로 사용자 승인 없이 자동으로 진행)

3. **전략 수정 시 플로우**
   - 전략 수정이 필요하면 `register_strategy_to_nest` 도구로 전략을 갱신하고,
     Hawk Picker가 새로운 종목을 선정한 뒤 Meerkat이 타점을 계산합니다.
   - 별도로 `meerkat_scanner`를 직접 호출하지 마세요.
     전략 등록 도구가 Hawk Picker를 자동으로 연결합니다.

4. **요약 보고 (텔레그램 알림)**
   - 전략이 수정되거나 종목/타점이 변경되면 시스템이 자동으로 텔레그램 알림을 전송합니다.
   - 변경된 사항(전략 수정 여부, 수정 이유, 종목 변경, 타점 갱신 내역)을
     상세한 메시지 형태로 출력하세요. 이 메시지가 텔레그램으로 사용자에게 전달됩니다.
   - 전략 수정이 없었다면 "오늘은 전략 변경 없음, 기존 타점만 갱신"과 같은 간략한 메시지를 출력하세요.
